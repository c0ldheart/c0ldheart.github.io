ZCMU1880 wjw的火车站(栈的基本应用)

ZCMU1880 wjw的火车站

Description

wjw最近新开了一座火车站…没错就是火车站，因为寒假过完同学们都该返校了，所以他准备大干一场，但是这里有一个问题，因为wjw的资金不足，所以这座火车站只有一条铁路，所有的火车从一侧进入，从另一侧出来，但是为了方便调度火车，所以wjw机智的修改了一下铁路。如下图，如果火车A首先进入铁路，然后火车B在火车A离开之前进入铁路，则火车A只有在火车B离开后才能离开。那么现在问题来了，有一串火车按给定顺序进入车站，wjw希望在通过他的一波操作使这列火车以另一个顺序开出火车站，但是他的智商并不支持他解决这个问题，所以你的任务是确定在给定进站顺序和出站顺序的情况下，给出调度操作。

Input

输入包含多组数据。每个测试数据包含一个正整数n表示火车数，接下去的两个序列表示进站顺序和出站顺序，火车编号为小写或大写字母，(a≠A）

Output

输出数据包含一个字符串“Yes.”或“No.”，表示是否有可行的调度方案，若有，则输出调度操作。

Sample Input

3 ABC CBA

3 abc cab

Sample Output

Case #1: Yes.

in

in

in

out

out

out

Case #2: No.

题意

虽然没看懂图，但通过样例基本了解了。每组样例给出两个字符串，第一个为还未进车站的顺序，第二个为希望的出站时的顺序，火车进站后可以选择直接出站，也可以选择暂时不走（压入站顶），等待它后面的火车做出同样的选择（进站后出站或在站顶等待），通过以上操作判断最终能否实现期望出战顺序。

思路

题意中的站就是栈，这是一题栈的应用。利用数组ans[ansSum]保存输出“in”还是“out”的数据，数据压入栈时同时判断它做出的选择，一旦发现能出栈，就要不停出栈直到不能出，出栈时答案数组保存一个“out”。最后判断栈是否为空，即为答案的”Yes“或”No“。

关键部分分为以下步骤：

1.for循环i读入字符串第i个字符，将他压入栈，ans数组保存一个”in”

2.判断是否能出栈、输出、ans数组保存信息

3.判断栈是否为空（火车是否成功按期望走出）
